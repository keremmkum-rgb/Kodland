# Kodland

## Projem için fikir:
> # ♟️ Satranç Efsaneleri

Satranç Efsaneleri, dünyanın en ünlü satranç oyuncularını tanıtan basit ve modern bir web sitesidir.

## Özellikler

* 🎯 Kullanımı kolay arayüz
* 📱 Mobil uyumlu tasarım
* 👑 Ünlü satranç oyuncularının tanıtımı
* 🖼️ Oyuncu fotoğrafları
* 🔍 Oyuncular hakkında kısa bilgiler

## Kullanılan Teknolojiler

* HTML5
* CSS3

## Kurulum

1. Bu depoyu klonlayın:

```bash
git clone https://github.com/kullaniciadi/satranc-efsaneleri.git
```

2. Proje klasörüne girin:

```bash
cd satranc-efsaneleri
```

3. `index.html` dosyasını tarayıcıda açın.

## Proje Yapısı

```text
satranc-efsaneleri/
│
├── index.html
├── magnus.jpg
├── kasparov.jpg
├── nakamura.jpg

```
